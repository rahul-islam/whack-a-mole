#### Pre-requisite
1. [NodeJS: >= v8](https://nodejs.org/dist/latest-v8.x/)
2. [Yarn](https://yarnpkg.com/en/docs/install)

#### How to run server
1. Install all dependencies

> `$  npm install`

2. Set environment

> `$  export NODE_ENV=production`

 To run server wth mock devices:

> `$  export NODE_ENV=development`

3. Start server

> `$  npm start`
